The Props List

Theater Props Management Tool

Everything you need to manage your theater props like a pro.

Get Started

Create Your First Show

Start by adding a show to organize your props and tasks.

  1. Click "Show Management" in the sidebar
  2. Click "Add New Show"
  3. Enter show name, dates, and venue
  4. Save your show
The Show Management dashboard where you can create and manage your productions

Add Props

Track every prop in your production.

  1. Go to "Props Inventory"
  2. Click "Add Prop"
  3. Fill in prop details (name, description, condition)
  4. Assign to a show
The Props Inventory page where you can add and manage all your props

Props Management

Adding Props

Learn how to add and manage props effectively.

  • Use descriptive names and detailed descriptions
  • Set appropriate status (in-development, ready, needs-repair)
  • Add photos for visual reference
  • Assign props to specific shows
The Add Prop form where you can enter detailed information about each prop

Prop Status

Track the lifecycle of your props.

  • In Development: Prop is being created or modified
  • Ready: Prop is complete and ready for use
  • Needs Repair: Prop requires maintenance
  • Retired: Prop is no longer in use
The Prop Detail page showing status, photos, and maintenance information

Shows Management

Creating Shows

Set up shows to organize your productions.

  • Add team members and assign roles
  • Set up packing lists for different locations

Packing Lists

Creating Packing Lists

Organize props for different locations and scenes.

  • Create containers for different locations (stage, storage, etc.)
  • Add props to containers with quantities
  • Generate QR codes for easy scanning
  • Share packing lists with team members

Task Management

Using Task Boards

Manage production tasks with drag-and-drop boards.

  • Create columns for different task statuses
  • Add tasks with descriptions and due dates
  • Assign tasks to team members
  • Drag tasks between columns to update status
The Task Board interface with drag-and-drop functionality for managing production tasks
