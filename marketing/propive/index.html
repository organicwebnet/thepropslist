PROPIVE

Free project logging tool for prop makers

What is PROPIVE?

PROPIVE is a simple, free project logging app for individual prop makers. Track your projects, log time, and organize files with Google Drive integration.

Key Features

  • Project Logging: Log your prop-making projects with detailed notes and progress tracking
  • Time Tracking: Track time spent on each project and task
  • Google Drive Integration: Organize your project files using Google Drive
  • Progress Tracking: Monitor project milestones and completion status
  • File Organization: Automatically organize project files in Google Drive
